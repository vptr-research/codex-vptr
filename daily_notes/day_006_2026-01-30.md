## Implementation
**File:** `projects/libft/src/ft_isdigit.c`

Logic:
- Check if c in range [48-57] ('0'-'9')
- Return 1 if true, 0 if false
- Uses decimal values (consistent with ft_isalpha decision)
- Handles EOF (-1) correctly by design (fails range check)

**Decision:** Maintained decimal values (48, 57) instead of char literals 
for consistency with ft_isalpha implementation style.
